# Kiosk_JAVA
2021 - 객체지향프로그래밍 TermProject

## **프로젝트 설명**

### ⭐ 스터디카페 키오스크

▶️ **입실과 퇴실**

- 로그인 필요 ( 회원이 아닐 경우 회원가입 필요 )
- 입실 : 로그인이 되면 빈 좌석 중 원하는 좌석을 선택하여 입실
- 퇴실 : 로그인이 되면 본인의 좌석에서 퇴실처리가 되고, 그 좌석은 빈자리가 됨.

**▶ 음료 주문**

- 원하는 음료 최대 5가지를 선택할 수 있음.
